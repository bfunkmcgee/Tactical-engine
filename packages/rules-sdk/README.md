# rules-sdk

`rules-sdk` provides rules/content contracts and runtime helpers for game-specific rule sets.

## Public API stability

- `beta`: `ContentPack`, `contentIndex`, `RuleSet`, `hooks`, `scenario-runtime`
- `internal`: `contentPackValidation`

## Compatibility expectations

- Beta exports may evolve as canonical rules/content APIs settle.
- Internal exports are not covered by compatibility guarantees.
- Consumers should treat internal exports as temporary/testing-oriented.
